cheat sheet pour les nuls les commandes de raccourcis clavier

Qu'est-ce que les raccourcis clavier?


Principalement,es raccourcis clavier sont les meilleurs moyens de pianoter sur le clavier plus rapidement.
Ces fonctionnalités permettent aussi à l'utilisateur de se préserver de mouvements intempestifs.
en gros, ils vous font travailler plus vite!


Déplacements

cd : Se rend au dossier.
cd / : Se rend au dossier à partir de la racine (commence par un / ).
cd / : Se rend à la racine.
cd .. : Se rend au dossier parent (on peut faire ../../dossier/dossier, cela suit le chemin).
cd ~ : Se rend dans son dossier personnel
~ : équivaut à « /root/ » en utilisateur « root » (administrateur) ou « /home// » pour un « utilisateur normal ».
cd ~ : Se rend au dossier personnel de l'utilisateur indiqué


Listes et détails

ls ou dir : Liste le contenu d'un dossier.
ls -l : Liste un dossier en détails. (Souvent remplacé par l'alias ll) df : Donne l'espace libre et occupé des disques et partitions.
du -hs : Donne la taille d'un dossier (avec les dossiers enfants compris).
du -hs . : Donne la taille du dossier courant (avec les dossiers enfants compris).
ls -a : Liste le contenu d'un dossier, y-compris les fichiers cachés (dont le nom commence par un point).


Copier, déplacer, supprimer

cp : Copier.
mv : Déplacer ou renommer.
rm : Supprimer un ou des fichiers.
rm -Rf : Supprimer même les dossiers non vides.
rmdir : supprimer un ou des dossiers vides

Créer

mkdir : Créer un dossier pico ou nano : Édite un fichier en mode texte avec Pico (si le fichier n'existe pas il pourra le créer à la sauvegarde).
ln -s : Créer un lien symbolique.

Afficher

cat : Affiche le contenu du fichier dans la console en bloc.
less : Affiche le contenu du fichier dans la console avec défilement. Entrer q pour quitter.
more : Similaire à less. head : Affiche le début du fichier dans la console.
head -n 10 : Affiche les dix premières lignes du fichier dans la console.
tail : Affiche la fin du fichier dans la console.
tail -n 10 : Affiche les dix dernières lignes du fichier dans la console.
tail -f : Affiche toutes nouvelles lignes dans la console.


Fermer / Quitter / Déconnecter

exit
